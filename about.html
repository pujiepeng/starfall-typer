项目缘起

星辰坠落 (Starfall Typer) 诞生于一次课程作业——将打字练习与游戏机制结合，创造一个既有趣又实用的网页应用。太空主题的灵感来源于对科幻电影和宇宙探索的热爱。

技术亮点

  • 纯 HTML/CSS/JS 实现，零框架依赖
  • Canvas 粒子特效系统
  • Web Audio API 音效生成
  • 响应式设计，适配手机与桌面
  • GitHub Pages 一键部署

技能标签

HTML5 CSS3 JavaScript Canvas Web Audio API 响应式设计 Git GitHub Pages 游戏开发 前端开发

学习历程

🌱

前端入门

阶段一

学习 HTML 基础标签、CSS 布局与样式、JavaScript 基本语法，掌握网页三件套的核心用法。

🚀

进阶学习

阶段二

深入学习 Canvas 绘图、DOM 操作、事件处理、动画循环。了解 Git 版本控制与 GitHub 协作流程。

⭐

项目实战

阶段三

独立完成个人网站设计与开发，包括打字游戏、多板块布局、子页面导航，并部署到 GitHub Pages 上线。

🎯

未来规划

阶段四

学习 React/Vue 等前端框架，掌握 Node.js 后端开发，向全栈工程师方向迈进。
